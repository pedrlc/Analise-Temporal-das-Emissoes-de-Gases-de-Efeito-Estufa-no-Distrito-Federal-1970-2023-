# Análise de Emissões de Gases de Efeito Estufa no Distrito Federal

Este projeto apresenta uma **análise exploratória e descritiva** das emissões de gases de efeito estufa (GEE) no Distrito Federal, utilizando **Python** e **Jupyter Notebook**.  
O estudo busca identificar **tendências temporais**, **picos de emissão** e **possíveis fatores associados**, com base em dados reais.

O projeto possui caráter **educacional e analítico**, sendo desenvolvido com foco em boas práticas de análise de dados e documentação.

---

##  Objetivos

- Analisar a evolução das emissões de gases de efeito estufa ao longo dos anos  
- Identificar períodos de aumento ou redução significativa das emissões  
- Explorar padrões setoriais e temporais  
- Praticar análise de dados, visualização gráfica e interpretação estatística  
- Documentar hipóteses e conclusões de forma clara e técnica  

---

##  Tecnologias Utilizadas

- **Python**
- **Jupyter Notebook**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**

---

## Principais Análises Desenvolvidas

- Evolução temporal das emissões de GEE

- Comparação entre setores emissores

- Identificação de picos e tendências

- Análise gráfica e interpretação contextual   

---
##  Como Executar o Projeto   

O projeto utiliza bibliotecas amplamente adotadas no ecossistema Python para análise de dados e visualização.
Instale cada uma delas individualmente com os comandos abaixo:   

```bash
pip install pandas   

pip install numpy   

pip install matplotlib   

pip install seaborn   

pip install jupyter
```    

Essas bibliotecas são responsáveis pela manipulação dos dados, cálculos estatísticos, geração de gráficos e execução do notebook.

---
##  Sobre os Dados

Os dados utilizados neste projeto são provenientes de bases públicas oficiais e contêm informações anuais sobre emissões de gases de efeito estufa no Distrito Federal, organizadas por setor e período.  
O conjunto de dados foi utilizado exclusivamente para fins acadêmicos e de análise exploratória.   

---     

##  Principais Resultados

A análise indica variações relevantes nas emissões ao longo dos anos, com destaque para períodos específicos associados a fatores estruturais, energéticos e eventos de grande impacto.  
Os resultados detalhados e as interpretações completas estão documentados no notebook.
